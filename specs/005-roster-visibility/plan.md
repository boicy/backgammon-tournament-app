# Implementation Plan: Roster Visibility

**Branch**: `005-roster-visibility` | **Date**: 2026-03-06 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `specs/005-roster-visibility/spec.md`

## Summary

Player names are invisible when the roster is expanded because `.live-roster__row` and `.live-roster__name` have no CSS rules, and the `.live-roster-inner` card wrapper is defined in CSS but never used in `rosterListHtml()`. Fix: wrap roster content in `.live-roster-inner` and add the missing CSS rules.

## Technical Context

**Language/Version**: Vanilla JavaScript ES2022+
**Primary Dependencies**: None (production)

**Testing**: Vitest 3.x (unit) + Playwright (e2e)
